1. нужно создать шаблон для разметки сетки с товарами магазина
2. у карточки товара должно быть описание и кнопка "Добавить в корзину"
3. При загрузке страницы нужно выполнить запрос на бекенд и полученный результат(массив с объектами товаров) отрендерить на странице
4. При нажатии на кнопку  "Добавить в корзину" весь товар нужно скрыть и отрендерить модалку для добавления товара.
5. На модалке нужно отрендерить имя товара, цену, поле для ввода количества и кнопку "Купить"
6. При клике на кнопку "Купить" объект с параметарми выбранного товара нужно записать в массив в localStorage информация из которого будет использована для рендера корзины.
7. В хедер добавить кнопку "Корзина".
8. Когда пользователь добавил хотя бы один товар на кнопке с корзиной должен повиться маркер с количеством наименований товара выбранных пользователем - https://i.imgur.com/KxHX4Us.png
9. Когда пользователь кликнет на корзину, нужно отрендерить модалку оформления заказа, в которой будет перечень всех товаров с их количетсвом и кнопками для возможности изменить количество или удалить товар из списка. Также добавить поле "Сумма заказа" в котором выводить сумму заказа.
10. В модалке оформления заказа нужно добавить кнопку "Оформить заказ", которая соберет все данные в объект -
{
    "order": [
        {
            "id": 25,
            "name": "TV",
            "price": 2500,
            "currency": "UAH",
            "quantity": 1
        },
         {
            "id": 11,
            "name": "Smartphone",
            "price": 2500,
            "currency": "UAH",
            "quantity": 1
        },
         {
            "id": 3,
            "name": "PC",
            "price": 3500,
            "currency": "UAH",
            "quantity": 3
        }
    ]
}
11. После того, когда заказ будет записан на бекенде и будет получен респонс от сервера в виде объекта с закозом и присвоенным ему айдишником, нужно скрыть модалку заказа и отренедрить модалку для оплаты заказа в которой будет поле для ввода суммы и кнопка оплатить.
13. При клике на кнопку оплатить на бекенд нужно выполнить PATCH запрос который допишет в данном заказе поле - isPayed: true
